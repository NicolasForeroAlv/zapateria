# Instrucciones para ejecutar Zapatería App

1. Instalar dependencias:
```
npm install
```

2. Ejecutar en modo desarrollo (ejemplo con nodemon/ts-node, ajustar según el proyecto):
```
npm run dev
```

3. Abrir en el navegador:
```
http://localhost:3000
```

Notas:
- Se han añadido las rutas `/api/products`, `/api/cart`, `/api/cart/add`, `/api/cart/remove` y `/api/cart/total`.
- Los archivos de datos temporales están en `src/data/data.json`.
